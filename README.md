# Analyse de journeaux de connexion

##Présentation du projet

Ce projet consiste à analyser des évènements de connexion à l'aide de python et de pandas. L'objectif est d'identifier:
- les connexions réussies
- les tentatives de connexion échouées
- les adresses IP les plus fréquentes
- les activités potentiellement suspectes

##Technologies utilisées

 Python; pandas; fichiers csv

 ##Jeu de données

 Un jeu de données fictif a été créé dans le cadre d'un exercice d'apprentissage. Chaque évènement contient:
 - la date et l'heure de connexion
 - le nom d'utilisateur
 - l'adresse IP utilisée
 - le statut de la connexion ( rééussie ou échouée)

 ##Les résultats obtenus

 L'analyse nous permet d'obtenir
 - le nombre total d'évènements
 - le nombre de connexions réussies
 - le nombre de connexions échouées
 - les adresses IP les plus fréquentes

 ##Compétences mises en oeuvre

 - Analyse de données
 - Utilisation de python
 - Traitement de fichiers csv
 - Analyse orientée cybersécurité
 - Documentation technique
